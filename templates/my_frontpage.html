Welcome to the GEO Knowledge Hub!!

An open-source digital repository for the knowledge created by the Group on Earth Observations.

Discover

As a user, find open, linked and curated knowledge packages containing all relevant resources necessary to reuse applications created within the GEO Work Programme activities and promote Open Science.

Share

As a provider, you can use the GEO Knowledge Hub to register your EO application and make it accessible to the whole community in just a few steps!

Contribute

The GEO Knowledge Hub is based on InvenioRDM and is fully open-source. Check the code on Github. Feel free to raise issues or contribute to the code following the contributors guidelines.
